## KEY CONCEPTS

### What Are Tokens?

Tokens are digital assets built on blockchain technology. Unlike cryptocurrencies (coins) which have their own dedicated blockchain, tokens are built on existing blockchains like Ethereum. [Source S: BitDegree]

A crypto token is "a digital asset built on blockchain technology" that offers "a wider range of functionalities within the decentralized world" beyond just transactions. [Source S: BitDegree]

Crypto tokens can represent:
- Digital certificates of ownership for real-world assets like property or artwork
- Ownership stakes in a project (similar to holding stock)
- Voting rights within a platform or blockchain network
- Access to services within decentralized applications (dApps) [Source S: BitDegree]

The difference between coins and tokens: A crypto asset with a dedicated blockchain is technically a coin, while assets created on a third-party blockchain are called tokens. [Source S: Ledger]

---

### Fungible vs Non-Fungible Tokens

**Fungible Tokens:**
"A fungible token is an asset that represents a fixed value and is often used as a means of payment or to access services within an ecosystem. Each token has the same value and is interchangeable 1:1 with the same type of asset." [Source S: FINST]

"Fungible means that you can interchange one unit of a token with any other unit of that token because they represent the same value." [Source S: Ledger]

Examples: cryptocurrencies like Bitcoin, stablecoins like USDC, utility tokens like UNI or LINK. One UNI token is always worth the same as any other UNI token. [Source S: Ledger]

**Non-Fungible Tokens (NFTs):**
"Each NFT is distinct and can't be exchanged on a one-to-one basis with another NFT." [Source S: BitDegree]

"Non-fungible tokens are unique digital assets that represent ownership of a specific item, such as digital art, collectibles, or in-game assets. They use blockchain technology to verify authenticity and ownership." [Source S: BitDegree]

Key differences:
- While you can freely trade fungible tokens like Bitcoin, NFTs are distinct and cannot be traded as easily due to their uniqueness
- Each NFT has its own unique identifier and characteristics
- NFTs cannot be divided into smaller pieces like cryptocurrencies (you can buy 0.5 BTC but not half an NFT) [Source S: Reddit]

### Token Standards

Token standards are technical rules that define how tokens work on a blockchain. On Ethereum, these are called ERC (Ethereum Request for Comments) standards.

**ERC-20: The Fungible Token Standard**
"First implemented in 2015, ERC20 is the token standard that allows developers to create fungible tokens for their Ethereum-based applications or protocols." [Source S: Ledger]

ERC-20 "defines a standard interface for fungible tokens, including functions like transfer, balanceOf, and approve. Allows tokens to be easily integrated into wallets, exchanges, and other applications." [Source S: Fiveable]

**ERC-721: The Non-Fungible Token Standard**
"ERC721 standard allows you to create non-fungible tokens—tokens that each possess a unique value and act as verifiably unique digital objects that cannot interchange with each other like ERC20 tokens." [Source S: Ledger]

"Unlike ERC20s, ERC-721 tokens are NFT tokens or not fungible." [Source S: Bit2Me Academy]

**ERC-1155: The Semi-Fungible Token Standard**
This standard "enables Ethereum developers to create fungible, semi-fungible, and non-fungible tokens using the same standard." [Source S: Ledger]

Using ERC-1155, developers "can create a single contract to support multiple types of individually configurable tokens," reducing complexity. [Source S: Ledger]

Example use case: A game developer can create one fungible token as in-game currency AND multiple non-fungible tokens for unique items (skins, weapons, merchandise) using one contract instead of multiple. [Source S: Ledger]

## EXAMPLES MAYA WOULD UNDERSTAND

### Amusement Park Analogy
"Think of them as digital keys that unlock features within a specific digital ecosystem... The type of token determines the specific 'ride' or feature it grants access to within the playground." [Source S: BitDegree]

### Vending Machine Analogy
Smart contracts work like a vending machine: "you insert the exact amount of money (representing crypto tokens) and press a button for your desired item. Then, the vending machine (acting like a smart contract) automatically verifies your payment and dispenses the item based on predefined rules." [Source S: BitDegree]

### Cookie Recipe Analogy
"Think of ERC-20 as a recipe for creating cookies. This token provides the technical foundation (the recipe) for creating tokens. Then, ICOs use ERC-20 tokens as a fundraising tool (selling the cookies)." [Source S: BitDegree]

### Real-World Asset Tokenization (Relevant to Regeneration)
- **Carbon credits**: Tokenization "involves converting ownership rights of an asset into a digital token on a blockchain. These tokens, representing specific asset values, can be traded or transferred digitally without intermediaries." [Source S: Osler]
- "Asset tokenization applies to various assets, including real estate, art, commodities and financial instruments." [Source S: Osler]
- "These assets can range from tangible items like real estate and commodities to intangible assets like intellectual property and carbon credits." [Source S: Zoniqx]
- Platforms like Carbonmark, EcoRegistry, and Puro.earth are "leading this shift, enabling real-time credit" tracking. [Source S: Carbonmark]

### Gaming/Collectibles
"CryptoKitties is one of the popular examples of NFT collections. Every NFT represents a unique digital cat with its own set of characteristics, like fur color, eye shape, and even special traits. These traits determine the asset's rarity and value." [Source S: BitDegree]
