# Research: What You Can Do on Ethereum

## Overview

Ethereum is an open, public blockchain launched in July 2015 by Vitalik Buterin and a small team of co-founders. It extends Bitcoin's capability of sending digital money by introducing **smart contracts**—open-source programs that let anyone create digital assets and decentralized applications (dapps) that run 24/7, globally [Source: ethereum.org].

Today, Ethereum hosts millions of users holding billions of dollars in assets, transacting trillions annually—all without a bank [Source: ethereum.org].

---

## Core Functions of Ethereum

### 1. Send and Receive Digital Money (ETH)

- **Ether (ETH)** is Ethereum's native cryptocurrency
- Send to anyone, anywhere in the world in seconds for a few cents
- Used to pay transaction fees (gas fees) on the network
- ETH is **deflationary**—a portion is burned with every transaction, permanently removing it from supply [Source: ethereum.org]

### 2. Use Smart Contracts

Smart contracts are self-executing programs stored on the blockchain that automatically enforce agreements when conditions are met. They:

- Run 24/7, 365 days a year
- Are accessible to anyone with an internet connection
- Require no intermediaries (banks, lawyers, notaries)
- Are transparent and tamper-proof [Source: ethereum.org]

### 3. Access Decentralized Finance (DeFi)

- **Lending and borrowing** — Earn interest on deposits or borrow assets without banks
- **Trading** — Swap tokens instantly on decentralized exchanges
- **Stablecoins** — Use crypto pegged to fiat currencies (e.g., USDC, USDT)
- **Yield farming** — Maximize returns by moving assets across protocols

### 4. Own Digital Collectibles (NFTs)

- **Non-fungible tokens (NFTs)** represent ownership of unique digital items
- Includes art, collectibles, gaming items, domain names
- Verified authenticity and ownership on-chain
- Cannot be copied or confiscated [Source: ethereum.org]

### 5. Use Decentralized Applications (dapps)

Dapps are applications built on Ethereum that operate without a central authority:

- **DeFi apps** — Compound, Aave, Uniswap
- **Gaming** — Axie Infinity, Gods Unchained
- **Social media** — Lens Protocol, Friend.tech
- **Marketplaces** — OpenSea, Blur
- **Payments** — Solana Pay integration, Shopify stablecoins [Source: ethereum.org]

### 6. Build and Deploy Applications

Developers can:

- Launch dapps with built-in global payments from day one
- Deploy tamper-proof smart contracts
- Create financial products anyone can build upon
- Example: PayPal launched its stablecoin PYUSD on Ethereum [Source: ethereum.org]

### 7. Stake ETH and Validate

Since the 2022 "Merge" upgrade, Ethereum uses **proof of stake**:

- Validators lock up ETH as collateral to process transactions
- Earn ETH rewards for honest validation
- Lose staked ETH if they act maliciously
- Requires 32 ETH to run a full validator node (or anyone can stake via pooling) [Source: ethereum.org]

### 8. Participate in Governance

- Vote on protocol changes via **Ethereum Improvement Proposals (EIPs)**
- Participate in DAO governance for protocols
- Contribute to the ecosystem (developers, node operators, community members)
- No central authority—changes require community support [Source: ethereum.org]

## Ethereum's Key Advantages

| Feature | Description |
|---------|-------------|
| **Censorship resistant** | No bank or corporation can block transactions; code enforces equal treatment |
| **Highly secure** | Attacking a single dapp requires compromising the entire network (billions in cost) |
| **Durable and reliable** | Network keeps running even if some nodes go offline—perfect uptime |
| **Permissionless** | No ID, bank account, or approval needed—just a wallet and internet [Source: ethereum.org] |

---

## Key Facts

- **Launched:** July 2015
- **Founder:** Vitalik Buterin (published whitepaper in 2013)
- **Consensus mechanism:** Proof of stake (upgraded from proof of work in 2022)
- **Energy efficiency:** 99%+ more efficient after The Merge
- **Native token:** ETH (ether)
- **No CEO or central authority** — run by community of developers, validators, and users [Source: ethereum.org]
