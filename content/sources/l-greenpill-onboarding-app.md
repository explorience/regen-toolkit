# Greenpill Onboarding App

The Greenpill Onboarding App is an open-source application designed to help onboard new members to the Greenpill Network and local chapters.

## Overview

Developed by the Greenpill Dev Guild, this application provides a structured onboarding experience for new community members. It can serve as both a reference implementation and a starting point for custom onboarding solutions.

## Key Sections

- **User Flows**: Structured onboarding pathways
- **Educational Content**: Integrated learning modules
- **Progress Tracking**: Member journey tracking
- **Community Integration**: Connection to Greenpill chapters
- **Technical Architecture**: Full-stack web3 application

## When to Reference

- Building onboarding experiences for local initiatives
- Looking for UX patterns for member journeys
- Need technical examples of web3 onboarding
- Want to understand Greenpill's approach to onboarding

## Content Highlights

The app represents practical thinking about how to bring new people into regenerative communities. The user flows and content structure are as valuable as the code.

## Integration Notes

For Regen Toolkit content about onboarding and member journeys, this app provides a concrete example. Consider the patterns it uses when advising communities on their own onboarding.
