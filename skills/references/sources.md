## Citation Format

Use inline citations like `[Source A]` in article body. In frontmatter:

```yaml
sources:
  - ReFi DAO Local ReFi Toolkit
  - Greenpill Local Regen Guide
  - original
```

## Accessing Sources

- **Internal sources (B, C, P, Q, R, S):** Check vault/projects/regen-toolkit/ or ask for access
- **Public sources:** Check URLs above
- **For new articles:** Request source access from project lead

## Source Quality Guidelines

- **Primary (A, D-H, J-L, N, O):** Authoritative, well-maintained docs
- **Secondary (B, C, M, P, R):** Useful context, verify current
- **Tertiary (Q, S):** Use with caution, fact-check thoroughly
