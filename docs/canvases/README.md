## How to Use

### Opening Canvases

1. **In Obsidian:**
   - Open Obsidian with vault at repository root
   - Navigate to `canvases/` directory
   - Double-click any `.canvas` file to open
   - Start with `regen-toolkit-master.canvas` for navigation

2. **Outside Obsidian:**
   - Canvas files are JSON format
   - Can be viewed as JSON or converted to images
   - Export from Obsidian as PNG/PDF for sharing

### Navigation

- **Start with Master Canvas:** Use `regen-toolkit-master.canvas` as your entry point
- **Click Canvas Cards:** Click any canvas file card to navigate to that canvas
- **Use Back Button:** Return to master canvas using Obsidian's back navigation
- **Cross-References:** Canvas files reference each other and project files
